# Stellar Burgers

Учебный проект, типа освоение React'а. Яндекс.Практикум, да.

## Что это такое

Как бы приложение для заказа бургеров.

Есть конструктор бургеров. Есть личный кабинет. Есть лента заказов.

## Какие технологии использованы

- Typescript
- React Router
- React DnD
- Redux Toolkit
- WebSocket
- CSS Modules
- Jest
- Cypress
